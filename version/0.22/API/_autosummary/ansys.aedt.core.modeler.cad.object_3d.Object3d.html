Object3d#

class ansys.aedt.core.modeler.cad.object_3d.Object3d(primitives, name=None)#

Manages object attributes for the AEDT 3D Modeler.

Parameters:
primitivesansys.aedt.core.modeler.cad.primitives_3d.Primitives3D

Inherited parent object.

namestr

Examples

Basic usage demonstrated with an HFSS design:

>>> from ansys.aedt.core import Hfss
>>> aedtapp = Hfss()
>>> prim = aedtapp.modeler

Create a part, such as box, to return an ansys.aedt.core.modeler.cad.object_3d.Object3d.

>>> id = prim.create_box([0, 0, 0], [10, 10, 5], "Mybox", "Copper")
>>> part = prim[id]
