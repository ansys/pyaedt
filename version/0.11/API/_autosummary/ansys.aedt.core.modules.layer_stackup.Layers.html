Layers#

class ansys.aedt.core.modules.layer_stackup.Layers(modeler, roughnessunits='um')#

Manages stackup for the Circuit and HFSS 3D Layout tools.

Parameters:
modeleransys.aedt.core.modeler.modeler_pcb.Modeler3DLayout
roughnessunitsstr, optional

Units for the roughness of layers. The default is "um".

Examples

>>> from ansys.aedt.core import Hfss3dLayout
>>> app = Hfss3dLayout()
>>> layers = app.modeler.layers
