Ports
=====
These classes are the containers of ports methods of the EDB for both HFSS and Siwave.


.. code:: python

    from pyaedt import Edb
    edb = Edb(myedb, edbversion="2023.1")

    # this call returns the EDB excitations dictionary
    edb.ports
    ...
