set_djordjevic_sarkar_model#

Material.set_djordjevic_sarkar_model(dk=4, df=0.02, frequency=1000000000.0, sigma_dc=1e-12, freq_hi=159154940000.0)[source]#

Set Djordjevic-Sarkar model.

Parameters:
dkint, float, str, optional

Dielectric constant at input frequency.

dfint, float, str, optional

Loss tangent at input frequency.

frequencyint, float, optional.

Input frequency in Hz.

sigma_dcint, float, optional

Conductivity at DC. The default is 1e-12.

freq_hiint, float, optional

High-frequency corner in Hz. The default is 159.15494e9.

Returns:
bool

True if successful, False otherwise.
