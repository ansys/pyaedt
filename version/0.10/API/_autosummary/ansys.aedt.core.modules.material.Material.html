Material#

class ansys.aedt.core.modules.material.Material(materiallib, name, props=None, material_update=True)[source]#

Manages material properties.

Parameters:
materiallibansys.aedt.core.modules.material_lib.Materials

Inherited parent object.

namestr

Name of the material.

props

The default is None.

material_updatebool, optional

The default is True.

Examples

>>> from ansys.aedt.core import Hfss
>>> app = Hfss()
>>> material = app.materials["copper"]
