PostProcessor3D#

class ansys.aedt.core.visualization.post.post_common_3d.PostProcessor3D(app)#

Manages the main AEDT postprocessing functions.

The inherited AEDTConfig class contains all _desktop hierarchical calls needed for the class initialization data _desktop and the design types "HFSS", "Icepak", and "HFSS3DLayout".

Note

Some functionalities are available only when AEDT is running in the graphical mode.

Parameters:
appansys.aedt.core.application.analysis_3d.FieldAnalysis3D

Inherited parent object. The parent object must provide the members _modeler, _desktop, _odesign, and logger.

Examples

Basic usage demonstrated with an HFSS, Maxwell, or any other design:

>>> from ansys.aedt.core import Hfss
>>> hfss = Hfss()
>>> post = hfss.post
