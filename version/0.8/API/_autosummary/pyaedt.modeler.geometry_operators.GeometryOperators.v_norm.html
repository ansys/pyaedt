static GeometryOperators.v_norm(a)[source]#

Evaluate the Euclidean norm of a geometry vector.

Parameters:
aList
List of ``[x, y, z]`` coordinates for the vector.
Returns:
float

Evaluated norm in the same unit as the coordinates for the input vector.
