Export layout
=============

Export layout.

The extension is accessible through the icon created by the Extension Manager in the **Automation** tab.

The following image shows the extension user interface:

.. image:: ../../../_static/extensions/export_layout_ui.png
  :width: 800
  :alt: Export Layout UI


The available arguments are: ``ipc_ui``, ``confg_ui``, and ``bom_ui``.
