EMI Heatmap
===========

The **EMI Heatmap** extension allows users to generate a 2D heatmap of EMI results between two radio bands.

The extension provides a Graphical User Interface (GUI) for configuration.

The following image shows the extension GUI:

.. image:: ../../../_static/extensions/emi_heatmap_ui.png
  :width: 800
  :alt: EMI Heatmap GUI


Features
--------

- View EMI results as a 2D heatmap between two selected radio bands.
- Export formatted results to Excel.
- Switch between light and dark themes in the GUI.


Using the extension
-------------------

1. Open the **Automation** tab in the EMIT interface.
2. Locate and click the **EMI Heatmap** icon under the Extension Manager.
3. In the GUI, users can select the tx/rx radio bands to compare using the dropdown menus.
4. Use the buttons at the bottom to:
   - Generate the results matrix
   - Export results to Excel
